# Personal Expense Tracker 💰
A console-based expense tracking application built in C++.

## Features
- Add expenses across 6 categories (Food, Transport, Shopping, Health, Billing, Education)
- View all expenses with a running total
- Filter expenses by category with subtotal
- Input validation for all user entries

## How to Run
1. Clone the repository
2. Compile: `g++ expense_tracker.cpp -o expense_tracker`
3. Run: `./expense_tracker` (Linux/Mac) or `expense_tracker.exe` (Windows)

## Built With
- C++ (Standard Library)
- Concepts: Structs, Arrays, Loops, Input Validation
